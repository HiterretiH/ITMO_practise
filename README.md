# Проверка оформления ВКР

Веб-сервис для автоматической проверки структуры и форматирования выпускных квалификационных работ на соответствие требованиям Университета ИТМО.

---

## Быстрый старт

```bash
git clone https://github.com/HiterretiH/ITMO_practise.git
cd ITMO_practise
docker-compose up -d
```

После запуска откройте браузер и перейдите на `http://localhost:7272`

## Кратко о сервисе

- **Форматы:** `.doc` / `.docx`
- **Ограничение:** до 50 МБ
- **Проверка:** шрифты, отступы, поля, заголовки, нумерация, таблицы, рисунки, формулы, список литературы и многое другое
- **Результат:** детальный отчёт в браузере и PDF

---

## Технологии

- **Frontend:** Angular + PrimeNG
- **Backend:** Spring Boot
- **Контейнеризация:** Docker + Docker Compose
